    03 · Product method

    Start with the workflow, then choose the tools.

    Use AI for variable interpretation, rules for repeatability, and human review for context, risk and accountability.

    1. 01

      Frame

      +

      Define the user, decision and friction.

    2. 02

      Map

      +

      Capture inputs, handoffs, exceptions and tacit judgement.

    3. 03

      Architect

      +

      Assign each step to data, rules, AI or people.

    4. 04

      Prototype

      +

      Build the smallest complete path with real inputs.

    5. 05

      Evaluate

      +

      Test quality, effort, failure modes and review thresholds.

    6. 06

      Operationalise

      +

      Set ownership, monitoring and feedback loops.

    Responsible by design

    Set the boundaries before automating.

    • TraceabilityLink claims to sources and transformations.
    • UncertaintyRoute low-confidence cases to review.
    • AccountabilityName the owner, checkpoint and override.
    • SimulationUse synthetic audiences to test hypotheses — not replace research.
